HTML Sémantique vs Non-Sémantique

  • HTML Sémantique
  • HTML Non-Sémantique

HTML Sémantique

Les éléments sémantiques en HTML ont une signification claire et explicite. Ils aident à structurer le contenu d'un document de manière logique, améliorant ainsi l'accessibilité, le référencement SEO et la compréhension du code par les développeurs.

Exemples d'éléments sémantiques :

  • <header> : Définit une en-tête pour un document ou une section.
  • <footer> : Contient des informations de pied de page.
  • <article> : Représente un contenu autonome comme un article de blog.
  • <section> : Utilisé pour diviser un contenu en sections thématiques.
  • <nav> : Contient un ensemble de liens de navigation.
  • <aside> : Représente un contenu secondaire ou complémentaire.
  • <main> : Représente le contenu principal du document.

Exemple d'HTML Sémantique :

<header>
  <h1>Bienvenue sur mon site web</h1>
  <nav>
    <ul>
      <li><a href="#home">Accueil</a></li>
      <li><a href="#about">À propos</a></li>
    </ul>
  </nav>
</header>

<main>
  <section>
    <h2>Introduction</h2>
    <p>Ceci est un paragraphe d'introduction.</p>
  </section>
</main>

<footer>
  <p>© 2025 Mon Site Web</p>
</footer>

HTML Non-Sémantique

Les éléments non-sémantiques n'ont pas de signification explicite pour le contenu qu'ils contiennent. Ils sont principalement utilisés pour structurer ou styliser un document sans ajouter de sens contextuel particulier.

Exemples d'éléments non-sémantiques :

  • <div> : Conteneur générique utilisé pour la mise en page.
  • <span> : Utilisé pour styliser une portion de texte sans signification sémantique.

Exemple d'HTML Non-Sémantique :

<div class="header">
  <h1>Bienvenue sur mon site web</h1>
  <div class="nav">
    <ul>
      <li><a href="#home">Accueil</a></li>
      <li><a href="#about">À propos</a></li>
    </ul>
  </div>
</div>

<div class="main">
  <div class="section">
    <h2>Introduction</h2>
    <p>Ceci est un paragraphe d'introduction.</p>
  </div>
</div>

<div class="footer">
  <p>© 2025 Mon Site Web</p>
</div>
